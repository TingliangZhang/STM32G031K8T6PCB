## 晶振

晶振设计参考AN2867
$$
\mathrm{C}_{\mathrm{L}}=\frac{\mathrm{C}_{\mathrm{L} 1} \times \mathrm{C}_{\mathrm{L} 2}}{\mathrm{C}_{\mathrm{L} 1}+\mathrm{C}_{\mathrm{L} 2}}+\mathrm{C}_{\mathrm{S}}
$$
[ TST(台湾嘉硕) 32M ±10ppm 12pF ](https://item.szlcsc.com/510342.html)

求得负载电容为20pF
